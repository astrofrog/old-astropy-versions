Console¶

class astropy.units.format.console.Console[source] [edit on github]¶

Bases: astropy.units.format.base.Base

Output-only format for to display pretty formatting at the console.

For example:

>>> import astropy.units as u
>>> print(u.Ry.decompose().to_string('console'))
              m^2 kg
2.1...*10^-18 ------
               s^2
