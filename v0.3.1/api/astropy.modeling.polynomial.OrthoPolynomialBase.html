OrthoPolynomialBase¶

class astropy.modeling.polynomial.OrthoPolynomialBase(x_degree, y_degree, x_domain=None, x_window=None, y_domain=None, y_window=None, param_dim=1, **params)[source] [edit on github]¶

Bases: astropy.modeling.polynomial.PolynomialBase

This is a base class for the 2D Chebyshev and Legendre models.

The polynomials implemented here require a maximum degree in x and y.
