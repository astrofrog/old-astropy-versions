SIP¶

class astropy.modeling.polynomial.SIP(crpix, a_order, b_order, a_coeff={}, b_coeff={}, ap_order=None, bp_order=None, ap_coeff={}, bp_coeff={}, n_models=None, model_set_axis=None)[source] [edit on github]¶

Bases: astropy.modeling.Model

Simple Imaging Polynomial (SIP) model.

The SIP convention is used to represent distortions in FITS image headers. See [R7] for a description of the SIP convention.

Parameters:

crpix : list or ndarray of length(2)

CRPIX values

a_order : int

b_order : int

SIP order for second axis

coefficients for the inverse transform
